Shaai

A pluggable blogging framework

Shaai is designed to be pluggable in entirety. We don't want you to create a project solely for integrating blogs into your website. It should be as simple as installing a node module. Hence, Shaai is built upon a few basic constituents, which are discussed ahead.

Shaai derives its name from a Marathi/Hindi word शाई which means ink. It's building blocks are named in a similar fashion. We suggest you go through the nomenclature once so that it becomes easy to install things, but you can always skip and get started rightaway.

Nomenclature

The core library exports a set of functions to talk to your blog related data. Basic functions like getBlogs() can be used to fetch posts and render as required.

Building blocks

1. Core library

The core library exports a set of functions to talk to your blog related data. Basic function like getBlogs() can be used to fetch posts and render as required.

2. Scrolls

Scrolls are templates that render your data. @shaai/scroll-ink is one of our homemade scrolls. It provides a blog design using Shaai's design system and is a clean way to show your blog posts.

Get Started

With Shaai, you can create new blogs or integrate them into your existing websites. To begin in plain JavaScript run the following.

npm i --save @shaai/scroll-ink
To use with React, include the @shaai/react wrapper too.

npm i --save @shaai/scroll-ink @shaai/react
To use Shaai, you need to initialise it using a config object. This contains properties like source of data and the related keys.
You can render your data using templates called scrolls. These are essentially renderers for your data from Shaai.
npm install --save @shaai/scroll-ink
